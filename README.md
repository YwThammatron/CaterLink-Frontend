# CaterLink Presentation Frontend Section (2 Participants)

Tools and Languages

Frontend
- React framework with Vite
- HTML and Javascript
- TailwindCSS
- Shadcn and Lucide icon
- Axios library

Backend
- Node.js
- Supabase database

**Main Requirement**
- User can view restaurants, restaurant details, and reviews
- User can compare and plan restaurants his/her want to try
- User can view blogs written by either customer and restaurant
- User can search restaurant filtered by name, main category, event category, and food category
- User can signup and login as customer or restaurant via its portal
- Customer can make order (reserve catering) and payment to restaurant
- Customer can view his/her order cart and history
- Customer can favourite his/her restaurant in home page
- Customer can cancel order during process before payment
- Customer can review restaurant after order is complete
- Customer can write customer blog and will be showed at blog page
- Restaurant will have to create its account and input restaurant nessessary details (when signup)
- Restaurant can view and edit its profile or change restaurant category tag
- Restaurant can view orders from customer and accept or decline order
- Restaurant can add, edit, and delete its catering packages
- Restaurant can create promotion to its packages
- Restaurant can write restaurant blog and will be showed at blog page

Responsibility - Frontend Developer
- Work with other 1 participant in frontend team
- Develop webpages in all restaurant section and restaurant signup and login portal
- Use Shadcn component for simplicity and minimalistic design and Lucide for icons
- Use Axios library for HTTP request and response methods
- Adjust some UX/UI designs for logical and practical reason with designer

Remarks
- This entire project actually have 5 participants which 2 are frontend, other 2 are backend, and the other is project manager and UX/UI designer
